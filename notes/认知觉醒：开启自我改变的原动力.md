# 第一章 大脑

- 三重大脑：本能脑、情绪脑、理智脑
  - 本能脑和情绪脑：以为自己处于原始社会，为了生存节省能量，思考、锻炼等高耗能行为被视为威胁，目光短浅、即时满足成为天性

- 舒适区，拉伸区，困难区
  - 拉伸区：有成就有挑战，进步最快
  - 困难区：容易因畏惧而逃避

- 成长权重对比
  - 改变量 > 行动量 > 思考量 > 学习量

- 复利曲线，积累到拐点
  - 冷静面对前期缓慢的增长并坚持到拐点
  - 比如时常用英语“熏耳朵”的人，某一天突然发现好像都能听懂了

# 第二章 潜意识

- 消除模糊，制造清晰
  - 人们习惯在舒适区兜圈，重复已经掌握的事情，对真正的困难视而不见--潜意识操控，基因认为这样耗能低
  - 优秀的人更倾向做高耗能的事--学霸记录错题本，集中精力攻克；一般同学更喜欢重复已经掌握的部分

- 回避痛苦不会使痛苦消失，反而使其转入潜意识，变成模糊感觉
  - 真正的困难总比想的要小很多
  - 人们拖延、纠结、畏惧的原有往往不是事情本身有多难，而是内心的想法变得模糊

  > 记住，任何痛苦事件都不会自动消失，哪怕再小的事情也是如此。要想不受其困扰，唯一的办法就是正视它、看清它、拆解它、化解它，不给它进入潜意识的机会，不给它变模糊的机会；即使已经进入潜意识，也要想办法将它挖出来。所以，当你感到心里有说不清、道不明的难受的感觉时，赶紧坐下来，向自己提问。
  >
  > 到底是什么让自己烦躁不安？是上台演讲、会见某人，还是思绪纷乱？
  >
  > 具体是什么让自己恐惧担忧？是能力不足、准备不够，还是害怕某事发生？
  >
  > 面对困境，我能做什么？不能做什么？如果做不到或搞砸了，最坏的结果是什么？
  >
  > 一层层挖下去，直到挖不动为止。坦然地承认、接纳那些难以启齿的想法，让情绪极度透明。虽然直面情绪不会让痛苦马上消失，甚至短时间内还会加剧痛苦，但这会让你主导形势，至少不会被情绪无端恐吓。
  >
  > 恐惧就是一个欺软怕硬的货色，你躲避它，它就张牙舞爪，你正视它，它就原形毕露。一旦把它看得清清楚楚，情绪就会慢慢从潜意识中消散，你的生活将会舒畅无比。

- 凭感觉-察觉潜意识发出的信息
  - 潜意识处理信息速度远高于意识，更容易察觉到一些信息

- 凭感觉学习
  - 一眼看出答案的，不用理它
  - 一眼看去就头痛，不知道在说什么，不用理它
  - 大致看出点思路，又要动点脑筋的，多做，这就是中间地带，拉伸区，进步最快的地方

- 先用感性选择，再用理性思考
  - 小事听从你的大脑，大事听从你的心

- 如何捕捉感性
  - 最：眼前一亮、心中泛起波澜的人和事；灵光乍现的想法；遭遇的痛苦等。捕捉它们深入分析挖掘
  - 总：脑子里总是不自觉跳出来的重复想法，心里总是挥之不去的事
  - 第一反应：对某人、某事，心中出现的瞬间反应或第一个念头
  - 梦境：潜意识已经捕捉到的信息，借助梦境提示
  - 身体：生理、心理不适，都会通过身体反映出来
  - 直觉：给一些无法解释、来路不明的信息开绿灯

# 第三章 元认知
  
- 反思能力：对我们的思考过程进行认知和理解
  - 观察自己的思维活动，找出其中不合理的地方，加以改进

- 监控自己的注意力，然后将其集中到需要关注的事情上来

- 需要选择的时候，停留几秒思考一下，审视当前思维，或许会做出不一样的选择

- 想好再做
  - 关键时间节点，花时间想好做什么
  - 列出所有想做的事，重要性、长远意义和内在动机，明确下一步做什么，只保留一个选择
  - 不要一丁点思考的时间都不花，饱和的行动自我感动自我满足

- 开船
  - 当舵手：掌握方向
  - 当水手：在甲板上忙个不停，船飘到哪里算哪里

# 第四章 专注力

- 做A想B
  - 注意力分为行动和感受两个部分
  - 例如，跑步、吃饭、睡觉时，脑子在想别的事
  - 让感受回归行动，身心合一，专注当下，享受当下

- 正确的方法，大量的练习
  - 正确的方法
    - 有定义明确的目标：练琴例子
    - 练习时极度专注
    - 能获得有效的反馈：教练、书籍影像、他人交流、自我反思
    - 始终在拉伸区练习：已掌握的❌ 太难的❌

# 第五章 学习力

- 难度匹配
  - 最佳承受范围，难以坚持时主动降低难度和强度

- 像对待考试一样

- 深度学习
  - 阅读→笔记→实践→教授

- 反思生活

- 不要太追求形式上的完整

- 认知闭合
  - 面对模糊的问题，有寻求确认答案的欲望。如古代人不知道下雨的原因，就给出雷公电母等解释
  - 打卡坏处：学不到，假装一下；学到了，立即停止，失去动机

- 设下限，不设上限
  - 设定最低目标，完成最低目标后，再追求更高的目标

- 尽快产出，换取反馈
  - 学说话、学走路，不必先学拼音、语法或力学、肌肉控制
  - 跳过原理，直接实操

- 主动休息
  - 感觉注意力开始分散时主动休息，不必靠意志力痛苦坚持

# 第七章 情绪力

- 稀缺心态导致判断力下降
  - 每个人的头脑配置都差不多
  - 贫穷使人变笨：贫穷造成的稀缺心态占用注意力
  - 事实上任何压力都会占用注意力，影响判断：失业压力、考试期限、工作压力

- 提高心智带宽（升级头脑配置）
  - 环境觉知，对压力环境的思考，压力环境下保持远见
  - 目标觉知，少即是多，做重要的事情，从忙碌中解脱
  - 欲望觉知，避免同时做很多事，留足闲余，从容
  - 情绪觉知，不要在大喜大悲、愤怒、恐惧、担忧等极端情绪时做决定
  - 闲余觉知，过多的闲余影响效率，大量金钱、时间导致无聊空虚，适当自我设限 ❓

- 多视角看问题
  - 勤移动
    - 换他人视角
    - 焦虑、紧张时设想自己是局外人，第三视角
    - 假设处于10年后，未来视角
  - 善学习
    - 有时候看不到一些角度是因为学识不够，像高人学习
  - 要开放
    - 保持客观，不臆断
    - 不把假设当事实，在不确定他人想法的情况下发泄情绪
    - 不先入为主，不带主观色彩，不戴有色眼镜，先了解事实，了解对方想法
    - 对任何人，先假设他是正直、善良、诚实的
  - 寻帮助
    - 不要闷头苦想，借助他人视角
  - 多运动
    - 多巴胺
  - 常反思
    - 烦恼写下来，笔和键盘帮你跳出单一视角

- 内在需求
  - 关系：良好的人际关系
  - 能力：有独特的本领、技能，为他人带去独特价值
  - 自主：自己选择想做的事
    - 不想做但必须做
      - 悲观的事物背后肯定有乐观的一面
      - 我不是在做这件事，我是在做另外一件事：我不是在为老板做事，我只是在提升自己；我不是去见领导，我只是去见一个朋友...
    - 世界的意义是我们自己赋予的，为什么不赋予它有用有好玩的意义呢？我们的情绪和注意力是自由的

# 第八章 早冥读写跑

- 早起
  - 1.5小时快速眼动、1.5小时非快速眼动不断切换，前两个单位3小时高质量睡眠
  - 人在睡眠后的3小时、4.5小时、6小时、7.5小时这几个节点醒来，就会觉得神清气爽，精力充沛

- 冥想

- 阅读
  - 阅读是在和顶级专家交流谈话
  - 提高思考密度：阅读时会思考，日常生活大多遵循习惯而没有思考
  - 选书
    - 解决自己最迫切的问题
    - 兴趣，难度，需求
  - 阅读是为了改变
    - 阅读后，思考、实践、教授
  - 高阶
    - 阅读时产生关联想到其他知识，一定留意，记下来
    - 读写不分家，阅读后能够用自己的语言阐释

- 写作
  - 无论学什么，都要想清楚它究竟在讲什么，它的实际意义是什么，用自己的话讲出来
  - 费曼技巧：通过自己的语言，用最简单的话把一件事情讲清楚，最好让外行人也能听懂
  - 用简单的语言
  - 像聊天一样的方式写作，不要说教
  - 类比（比喻）：以一种事物认知另一种事物
